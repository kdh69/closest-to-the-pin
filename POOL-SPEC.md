# Level Design Spec

## Structure (all 20 levels identical)

- Position 1: easy tier (100 max pts)
- Position 2: easy tier (100 max pts)
- Position 3: medium tier (200 max pts)
- Position 4: hard tier (300 max pts)
- Position 5: hard tier (300 max pts)
- Max per level: 1000 pts

## Repetition Rules

- Each easy location: max 2 levels (most 1x, ~10 used 2x)
- Each medium location: max 1 level (20 of 23 used; 3 reserved for daily)
- Each hard location: max 2 levels (most 1x, ~6 unused for daily)

## Geographic Spread

- No two locations within 500 miles (800 km) in same level
- Forces global variety per level
- Check before locking each level — swap if too close

## Uniqueness Between Levels

- No two levels share 4+ locations
- Levels numbered 1-20 in grid order
- Identical difficulty floor across all 20 levels — no level harder than another

## Build Approach

1. List all 50 easy locations, plan capacity (most 1x, ~10 used 2x)
2. List all 46 hard locations, plan capacity (most 1x, ~6 unused for daily)
3. Pick 20 medium locations from 23 available, leave 3 unused
4. Build levels one at a time:
   a. Pick 2 easy with remaining capacity
   b. Pick 1 unused medium
   c. Pick 2 hard with remaining capacity
   d. Verify no two within 500 miles
   e. Verify no 4+ shared with prior level
5. Backtrack and swap if stuck

## Reporting (when done)

- Confirmed tier totals: easy=50, medium=23, hard=46
- All 26 new locations added with verified coordinates
- Table of all 20 levels with 5 locations at each position
- Repetition count per easy location
- Repetition count per hard location
- List of 20 medium locations used (plus 3 unused)
- Confirmation no level has two locations within 500 miles
- Confirmation no two levels share 4+ locations
